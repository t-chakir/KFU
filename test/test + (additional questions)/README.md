# Список оставшихся вопросов теста Чабанова

Дана лямбда-функция на языке С++. Какой её тип возвращаемого значения?
```cpp
[](int a, int b){return a > b;};
```

Ответ:
- [ ] double
- [x] bool
- [ ] Никакого, т.к. тип не задан явно
- [ ] int
