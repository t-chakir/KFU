# Список вопросов 6 теста Чабанова

Дан фрагмент кода на С++. Что будет напечатано в результате его выполнения?
```cpp
int i = 1;

namespace A {
    namespace B {
        int i = 2;
        void print() {
            std::cout << i;
        }
    }
}

int main() {
    int i = 3;
    A::B::print();
}
```

Ответ:
- [ ] 1
- [ ] Ошибка
- [x] 2
- [ ] 3
